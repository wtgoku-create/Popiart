## 常见使用场景

### 社交媒体内容
- **微信朋友圈 / 公众号**: `-a 1:1`、`-a 3:4` 或 `-a 9:16`，适合配图文案图、封面图
- **小红书笔记**: `-a 3:4` 或 `-a 4:5`，竖版内容更常用
- **抖音 / 视频号**: `-a 9:16`，适合竖版封面和内容图
- **B 站 / 知乎**: `-a 16:9 -r 2K`，适合视频封面、专栏头图
- **微博**: `-a 1:1` 或 `-a 3:4`，适合九宫格配图和话题图

### 电商与营销
- **淘宝 / 京东主图**: `-a 1:1`，适合白底产品图和场景图
- **电商详情页**: `-a 3:4`，适合产品展示、卖点图、对比图
- **活动海报**: `-a 2:3` 或 `-a 9:16`，适合促销海报和节日海报
- **PPT 配图**: `-a 16:9`，适合演讲背景和数据可视化插图

### 个人创作
- **艺术创作**: 可用于视觉信息图、笔记图、插画图等
- **头像 / 壁纸**: `-a 1:1` 适合头像，`-a 9:16` 适合手机壁纸
- **表情包 / 梗图**: `-a 1:1`，适合搞笑图片和表情包素材
- **节日贺卡**: `-a 3:4` 或 `-a 4:5`，适合春节、中秋等祝福图
- **LOGO / 图标**: `-a 1:1`，适合品牌标识和应用图标

### 编辑任务
- **风格转换**: “转换成国潮 / 水墨 / 二次元 / 像素风格”
- **元素添加**: “添加春节元素 / 红包 / 灯笼 / 烟花”
- **背景替换**: “将背景换成故宫 / 西湖 / 长城 / 现代都市”
- **色彩调整**: “调整为莫兰迪色系 / 中国传统色”
- **对象移除**: “去除水印 / 路人 / 杂物”
- **画质修复**: “修复老照片 / 提升清晰度 / 去噪点”
